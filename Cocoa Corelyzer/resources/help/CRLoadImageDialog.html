CRLoadImageDialog

Use this dialog to select the image files that you want to load into Corelyzer.

How to determine the "DPI" value

"DPI" stands for "dots per inch". Corelyzer use this value to map digitalized image to physical scale. DPI values are determined using the following order:
  • If the DPI values are not zeros in the table:
    • The DPI_x and DPI_y values will be used directly if they are not zeros.

  • If "DPI" values are zero, but the "length" values are not zero
    • The DPI values will be calculated using the dimensions of the image(width and height in pixels) and the physical (core) length of the image (in meter).

    dpi = (image_width_in_pixel) / (length * 100 / 2.54)
  • Default DPI values(254.0) will be used
